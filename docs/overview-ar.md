# مرسل البريد الإلكتروني والواتساب الجماعي - نظرة عامة

هذا التطبيق هو أداة قوية وسهلة الاستخدام لإرسال إشعارات جماعية مخصصة عبر البريد الإلكتروني والواتساب. تم تصميمه للتعامل مع قوائم المراسلة المستندة إلى ملفات CSV مع دعم للقوالب الديناميكية وتنسيقات RTL (من اليمين إلى اليسار) للغة العربية.

## الميزات الأساسية

### 1. إرسال الرسائل عبر قنوات متعددة

- **وضع البريد الإلكتروني:** إرسال رسائل بريد إلكتروني جماعية باستخدام أي مزود SMTP (iCloud أو Gmail أو مخصص).
- **وضع الواتساب:** إرسال رسائل واتساب مباشرة إلى أرقام هواتف المستلمين مع محتوى مخصص معبأ مسبقاً.

### 2. معالجة بيانات CSV

- **تحليل ذكي:** تحميل ملفات CSV مع دعم ترميز UTF-8 وعناوين الأعمدة باللغة العربية.
- **ربط الأعمدة:** ربط أعمدة ملف CSV بحقول محددة مثل الاسم، البريد الإلكتروني، أو رقم الهاتف.
- **معاينة البيانات:** عرض بياناتك مباشرة في لوحة التحكم قبل البدء بالإرسال.

### 3. التخصيص والقوالب

- **المتغيرات الديناميكية:** استخدم صيغة `{{اسم_العمود}}` لإدراج البيانات من ملف CSV في موضوع الرسالة ونصها.
- **دعم HTML:** إنشاء محتوى بريد إلكتروني غني باستخدام وسوم HTML (مثل `<h2>` و `<b>`).
- **دعم RTL:** تفعيل وضع "من اليمين إلى اليسار" لضمان المحاذاة المثالية للرسائل العربية.

### 4. نظام إرسال قوي

- **إعدادات SMTP:** تهيئة خادم البريد الخاص بك بشكل آمن مع دعم كلمات مرور التطبيقات (App-specific passwords).
- **تحديد السرعة:** ضبط تأخير مخصص بين الرسائل (بالثواني) لتجنب فلاتر البريد العشوائي أو قيود الخادم.
- **سجلات مباشرة:** مراقبة عملية الإرسال في الوقت الفعلي مع سجلات مفصلة للنجاح والأخطاء.

## خطوات العمل

1. **التحميل:** قم بإسقاط ملف CSV في منطقة التحميل.
2. **الربط:** اختر الأعمدة التي تحتوي على العناوين (البريد/الهاتف) والأسماء.
3. **الإنشاء:** اكتب قالب رسالتك باستخدام المتغيرات الديناميكية. قم بتكوين إعدادات SMTP أو تفضيلات الواتساب.
4. **الإرسال:** راجع الملخص وابدأ عملية الإرسال. تابع التقدم من خلال لوحة التحكم المباشرة.

## التفاصيل التقنية

- **الواجهة الأمامية:** Next.js (React)
- **التنسيق:** CSS3 مع دعم RTL
- **تحليل الملفات:** PapaParse
- **محرك البريد الإلكتروني:** Nodemailer (عبر مسارات API)
- **تكامل الواتساب:** واجهة برمجة تطبيقات `wa.me` من واتساب
